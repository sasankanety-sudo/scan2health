# Scan2Health

Scan2Health is a smart web app that helps users understand food and product health using AI.

## Problem
People don’t know whether products are healthy or harmful.

## Solution
Scan2Health analyzes ingredients and gives simple health insights.

## Features
- Product scanning
- Ingredient analysis
- Health score
- Easy UI

## Tech Stack
HTML, CSS, JavaScript
